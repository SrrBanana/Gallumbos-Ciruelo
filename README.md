Entrega EC1: Diseño web con HTML y CSS
“Gallumbos Ciruelo” – Línea de ropa interior masculina
Contexto
La empresa Gallumbos Ciruelo va a lanzar su primera página web corporativa para
presentar su línea de ropa interior masculina.

Tu trabajo consiste en crear la estructura HTML y aplicar los estilos CSS
necesarios para obtener una página moderna, clara y responsive, utilizando los
conceptos vistos en clase.
No se proporciona ningún código base.

Deberás decidir qué etiquetas HTML usar y qué estilos CSS aplicar.

Requisitos generales
• La web debe llamarse Gallumbos Ciruelo
• Separar correctamente en dos archivos:
o HTML → estructura
o CSS → presentación
• Usar al menos una imagen para el logo y una para algún producto.

Estructura de la página
La página debe contener como las siguientes secciones:

1️. Cabecera (Header)
Debe incluir:
• Nombre de la marca: Gallumbos Ciruelo
• Un eslogan o frase publicitaria (por ejemplo: “Comodidad que se lleva por
dentro”)
• Un logo o imagen representativa
Debes de decidir qué etiquetas HTML utilizar y cómo organizar el contenido.

2️. Menú de navegación
Secciones recomendadas:
• Inicio
• Productos
• Colección
• Contacto
• Sobre nosotros
Se valorará el uso correcto de listas y enlaces.

3️. Sección “Sobre Gallumbos Ciruelo”
Debe incluir:
• Un título de sección
• Un texto descriptivo explicando:
o Qué tipo de marca es
o A qué público va dirigida
o Qué la diferencia de otras marcas

Se valorará el uso de párrafos, énfasis en el texto y organización clara.

4️. Sección de productos / colección
Debe mostrar una colección de gallumbos.
Cada producto debe incluir:
• Nombre del modelo
• Breve descripción
• Precio
• (Opcional) Imagen del producto
Esta sección debe adaptarse al tamaño de pantalla (flex):
• En móvil: los productos se muestran uno debajo de otro
• En pantallas grandes: los productos se muestran en fila
5️. Sección destacada
Crear una caja destacada que resalte alguna ventaja de la marca, por ejemplo:
• “Algodón 100% natural”
• “Diseño ergonómico”
• “Fabricado en España”

Debe apreciarse claramente el uso de:
• margen
• padding
• borde
• fondo

6️. Formulario de contacto
Debe permitir al usuario enviar una consulta.
Campos mínimos:
• Nombre
• Correo electrónico
• Tipo de consulta (lista desplegable)
• Mensaje
• Botón de envío
Se valorará el uso correcto de etiquetas de formulario y su maquetación.

7️. Pie de página (Footer)
Debe incluir:
• Texto de derechos de autor
• Año actual
• Nombre de la marca

Requisitos de HTML
En el desarrollo de la práctica se deberán utilizar, cuando sea adecuado, las
siguientes etiquetas HTML:
Estructura y semántica
• <html>
• <head>
• <body>
• <header>
• <nav>
• <main>
• <section>
• <article>
• <footer>

Contenido textual
• <h1> (título principal de la página)
• <h2> (títulos de secciones y subsecciones)
• <p>
• <strong>
Enlaces y navegación (1 enlace suficiente)
• <a>
• <ul>
• <li>
Imágenes y recursos
• <img>
• <figure>
• <figcaption>
Contenedores genéricos
• <div>
• <span>
Se valorará el uso de etiquetas semánticas frente a contenedores genéricos cuando
sea posible.
Formularios
• <form>
• <label>
• <input>
• <select>
• <option>
• <textarea>
• <button>

Requisitos de estilo (CSS)
El archivo CSS debe incluir, al menos:
• Uso de variables CSS para:
o Colores principales
o Espaciados
• Aplicación del modelo de caja
• Uso de diferentes valores de display
• Al menos un ejemplo de position:
o relative, absolute o fixed
• Diseño responsive usando media queries
